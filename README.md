# 🛒 Simple Python Shop Billing System

यह एक सिंपल कमांड-लाइन (command-line) बेस्ड बिलिंग सिस्टम है जिसे Python में लिखा गया है। यह प्रोग्राम एक दुकान का मेनू दिखाता है, यूजर से आर्डर लेता है, और आख़िरी में टोटल बिल कैलकुलेट करके बताता है।

## ✨ Features
* **Interactive Menu:** दुकान के सभी आइटम्स और उनके दाम स्क्रीन पर दिखते हैं।
* **Continuous Ordering:** यूजर एक बार में कई (multiple) आइटम्स आर्डर कर सकता है।
* **Smart Input Handling:** यूजर अगर कैपिटल लेटर्स या एक्स्ट्रा स्पेस के साथ इनपुट दे, तो भी प्रोग्राम उसे समझ लेता है (`.lower()` और `.strip()` का इस्तेमाल)।
* **Real-time Bill:** हर आइटम ऐड होने पर अभी तक का टोटल बिल दिखाता है।
* **Item Validation:** अगर यूजर ऐसा आइटम मांगे जो लिस्ट में नहीं है, तो प्रोग्राम सही मैसेज दिखाता है।

## 🚀 How to Run

1. अपने कंप्यूटर में यह पक्का करें कि Python इनस्टॉल हो।
2. इस रिपॉजिटरी या फाइल (`shop.py`) को अपने सिस्टम पर डाउनलोड करें।
3. टर्मिनल (Terminal) या कमांड प्रॉम्प्ट (Command Prompt) ओपन करें।
4. उस फोल्डर में जाएँ जहाँ यह फाइल सेव है।
5. नीचे दिया गया कमांड रन करें:

```bash
python shop.py
```

## 🛠️ Logic & Approach
यह प्रोजेक्ट Python की Dictionary (मेनू स्टोर करने के लिए), Lists (कार्ट आइटम्स स्टोर करने के लिए), और `while` loop (लगातार आर्डर के लिए) के बेसिक कांसेप्ट का इस्तेमाल करता है।
